    그놈 한글화 프로젝트 #

    그놈 한글화 프로젝트는 그놈 프로젝트에서 개발하는 애플리케이션을 한글화하는 프로젝트입니다. 여러분이 그놈을 기반으로한 우분투 리눅스를 설치할 때, 한글을 선택하면 모든 메뉴가 한글로 나타나는데, 이 프로젝트의 결과물입니다. 현재 프로젝트 멤버는 다음 같습니다.

    • 감독자: 류창우(Changwoo Ryu)
    • 검토자: 조성호(Seong-ho Cho)
    • 번역
      • 맹진영(Jinyeong Maeng)
      • AlexKodala
      • 엄지용(Jeeyong Um)
      • 문관경(Gwan-gyeong Mun)
      • 신현수(Hyunsu Shin)
      • 성대현(DaeHyun Sung)

    아직도 한글화할 부분이 많습니다. 참여를 원하는 분은 아래 프로젝트 페이지를 방문하시기 바랍니다.

    • https://l10n.gnome.org/teams/ko/
